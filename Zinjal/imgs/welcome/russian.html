Создание простой программы
Для быстрого начала работы (без создания проекта и/или определения дополнительных параметров). Выбирайте шаблон и начните писать код.

Создание нового проекта
Для более сложных программ. Проект позволит вам создавать программы состоящие из нескольких файлов, устанавливать отличные (от предопределенных) параметры компиляции и опции среды, а также интегрировать различные внешние инструменты, и т.д.

Импорт существующего проекта
Используйте эту опцию для определения нового проекта ZinjaI из нескольких уже существующих файлов исходников.
Открытие существующего файла или проекта
  • Последние открытые файлы: (больше...)
    1. ${FILE:01}
    2. ${FILE:02}
    3. ${FILE:03}
  • Последние открытые проекты: (больше...)
    1. ${PROJECT:01}
    2. ${PROJECT:02}
    3. ${PROJECT:03}
